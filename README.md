# DecodeLabs Cloud Computing Internship - Week 1 Project

## AWS S3 Static Website Hosting

## Project Name
The Global Launch - Cloud Computing Project

## Objective
To deploy a static portfolio website using Amazon S3 Static Website Hosting and make it accessible through a public website URL.

## Technologies Used

- Amazon Web Services (AWS)
- Amazon S3
- HTML
- CSS

## Project Description

In this project, I created a cloud-hosted portfolio website and deployed it using Amazon S3 static website hosting.

The website contains information about my:
- Profile
- Technical skills
- Cloud projects
- Certifications
- Contact details

## Implementation Steps

1. Created an AWS S3 bucket.
2. Enabled static website hosting.
3. Uploaded website files (`index.html` and `style.css`).
4. Configured bucket permissions for public access.
5. Generated the S3 website endpoint.
6. Verified the website deployment.

## AWS Service Used

### Amazon S3
Used for storing and hosting the static website files.

## Project Outcome

The portfolio website was successfully deployed on AWS S3 and accessed through the S3 website endpoint.
